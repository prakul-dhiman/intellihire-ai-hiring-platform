# 🚀 IntelliHire

### AI-Powered Technical Hiring Platform

IntelliHire is a modern **AI-driven recruitment platform** designed to transform the way companies evaluate technical candidates.

The system combines **real-time collaborative interviews, automated behavioral analysis, and AI-assisted coding assessments** to provide recruiters with deeper insights into candidate performance.

Built with a scalable **full-stack architecture**, IntelliHire enables seamless interaction between candidates and recruiters during live technical interviews.

---

# ✨ Key Features

## 🧠 AI Behavioral & Technical Analysis

### Automated Interview Evaluation

The platform analyzes candidate speech patterns during live interviews to detect **filler words ("um", "uh")**, evaluating communication fluency and clarity.

### Contextual AI Hints

Candidates receive **smart hints during coding assessments** without revealing the final solution, encouraging problem-solving rather than simple answer retrieval.

### Candidate Scoring Engine

A heuristic scoring system combines:

• Resume quality
• Coding performance
• Interview communication

to generate an **overall candidate score for recruiters**.

---

# 💻 Advanced Technical Assessment

## Collaborative Code Editor

IntelliHire integrates professional coding tools including:

• **Monaco Editor** (VS Code engine)
• **CodeMirror**

This allows candidates to code in a familiar environment during live interviews.

## Sandboxed Code Execution

Code is securely compiled and executed using the **Judge0 API**, supporting multiple programming languages including:

• Java
• Python
• JavaScript
• C++

---

# 🌐 Real-Time Collaboration

## Live Interview Rooms

Using **Socket.io**, the platform supports:

• Real-time collaborative coding
• synchronized communication between recruiter and candidate
• live interview sessions

## Integrated Messaging

A built-in chat system allows recruiters and candidates to communicate instantly during interviews.

---

# 📄 AI Resume Processing

IntelliHire automatically processes uploaded resumes using **pdf-parse**, extracting:

• skills
• work experience
• education

This enables automated resume scoring and faster candidate screening.
